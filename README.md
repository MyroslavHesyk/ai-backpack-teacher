# AI-рюкзак вчителя

**AI-рюкзак вчителя** — це інтерактивний освітній міні-сайт для педагогів, присвячений темі **«Штучний інтелект на допомогу вчителю»**.

Сайт створений як практичний онлайн-супровід майстер-класу для вчителів. Його мета — не просто розповісти про штучний інтелект, а допомогти педагогам одразу спробувати AI-інструменти в роботі: сформулювати промпт, створити план уроку, підготувати завдання, згенерувати ідеї для візуалів, адаптувати матеріали під різний рівень учнів і зібрати власний набір корисних інструментів.

---

## Для кого цей сайт

Сайт призначений для:

- учителів різних навчальних предметів;
- класних керівників;
- педагогів, які тільки починають знайомство зі штучним інтелектом;
- вчителів, які вже використовують ChatGPT, Gemini або інші AI-сервіси, але хочуть робити це системніше й ефективніше;
- учасників освітніх майстер-класів, тренінгів і літніх шкіл.

Матеріали подані простою мовою, без складної технічної термінології, з акцентом на практичне використання у школі.

---

## Мета сайту

Мета проєкту — показати, що штучний інтелект може бути для вчителя не заміною, а **помічником для рутинних, творчих і підготовчих завдань**.

Сайт допомагає педагогам зрозуміти:

- як правильно ставити завдання ChatGPT і Gemini;
- як писати якісні промпти;
- як уточнювати відповіді AI;
- як створювати плани уроків, тести, рубрики та завдання;
- як адаптувати матеріали для різних рівнів учнів;
- як використовувати AI для створення зображень, презентацій і навчального контенту;
- як відповідально застосовувати AI в освітньому процесі.

Головна ідея сайту:

> AI пропонує → вчитель перевіряє → вчитель адаптує → учень навчається.

---

## Що є на сайті

Сайт побудований як інтерактивна мандрівка з 12 станцій. Кожна станція присвячена окремому практичному аспекту використання штучного інтелекту в роботі вчителя.

### Основні блоки сайту

- **Вступ до AI для вчителя** — пояснення, чим AI може бути корисним у щоденній роботі педагога.
- **ChatGPT і Gemini** — короткий огляд найпопулярніших інструментів для роботи з текстами, ідеями, уроками й зображеннями.
- **Промптинг для вчителя** — пояснення, як писати детальні й ефективні запити до AI.
- **Промпт-конструктор** — інтерактивний блок, який допомагає зібрати готовий промпт під конкретний предмет, клас і тему.
- **Уточнювальні запити** — приклади команд, які допомагають покращити першу відповідь AI.
- **План уроку** — приклади використання AI для швидкого створення структури уроку.
- **Диференціація** — адаптація одного матеріалу для різних рівнів учнів.
- **Тести й рубрики** — створення запитань, завдань і критеріїв оцінювання.
- **AI-зображення** — правила генерації навчальних візуалів, постерів і картинок для уроків.
- **Комунікація** — допомога у створенні повідомлень для батьків, учнів і колег.
- **Відповідальне використання AI** — базові правила безпечного й етичного застосування штучного інтелекту.
- **Фінальний AI-пакет уроку** — генерація комплексного промпту для створення повного набору матеріалів до уроку.

## Головний принцип

Цей сайт не навчає «замінювати вчителя штучним інтелектом».  
Він показує, як зробити AI корисним помічником учителя.

Штучний інтелект може швидко створити чернетку, запропонувати ідеї, структурувати матеріал або допомогти з оформленням. Але саме вчитель залишається людиною, яка перевіряє, адаптує, пояснює, підтримує учнів і відповідає за якість навчання.
